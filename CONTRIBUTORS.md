# Contributing to Lektr Browser Extension

Thanks for your interest in contributing to Lektr! We welcome contributions from everyone.

## Getting Started

1. **Fork the repository** on GitHub.
2. **Clone your fork** locally.
3. **Install dependencies**:

   ```bash
   npm install
   ```

4. **Create a branch** for your feature or fix:

   ```bash
   git checkout -b feature/my-new-feature
   ```

## Development

- Run `npm run dev` to start the development server.
- Access the extension in Chrome at `chrome://extensions` by loading the unpacked extension from `.output/chrome-mv3-dev`.

## Pull Requests

1. Ensure your code follows the project's coding standards.
2. Update documentation if necessary.
3. Submit a Pull Request against the `main` branch.

## Code of Conduct

Please be respectful and kind to other contributors. Harassment of any kind will not be tolerated.
